# サイト設計図 (Implementation Plan) - 第2版

## 概要
大谷尚（おおたに ひさし）公式サイト。
**「優しさを連鎖させ、人と社会を未来につなげる」**という力強いコアメッセージを起点に、訪問者に知的で誠実な印象を与え、最終的にお問い合わせ等のアクションへと自然に繋げるサイトを目指します。
「初めての方でも着実に実装し、安全に運用・公開できる」ことを最優先に、Viteや不要なJavaScriptなどの複雑な環境を使わず、最もシンプルで堅牢なHTMLとCSSのみで構成しています。

## 1. サイトの目的・ターゲット
- **目的**: 弁護士および株式会社フクイト取締役としての「圧倒的な専門性と知的な経歴」、そして根底にある「福井が好き」「優しさを連鎖させる」という想いを伝え、共感した人に「講演・プロジェクトの相談などで対話してみたい」と感じてもらうこと。

## 2. 実装済みの機能・ページ構成（現在保存されている状態）
以下の5つのステップで情報を構成するシングルページ（縦長）構成です。

1. **Hero/Top (ファーストビュー)**: 
   - メッセージ：**「優しさを連鎖させ<br>人と社会を未来につなげる」**
   - デザイン：深く静謐なネイビーブルーのグラデーション背景。極太の白抜き明朝体テキストが浮かび上がる、非常に重厚で知性を感じさせるデザイン。

2. **Message (エッセイ)**:
   - 「福井が好き」という郷土への想いや、AIガバナンスを通じた社会貢献のビジョンを語りかけるエッセイ。
   - デザイン：文字を詰め込まず、詩を読むようなゆったりとした余白を持たせた白・ソフトブルー基調の空間。

3. **About (自己紹介・経歴)**: 
   - 肩書：「弁護士」と潔く記載し、圧倒的な信頼感を演出。
   - 略歴：フレックスボックスを用いて、年号と出来事を履歴書のように直感的に読みやすく美しく並べたリスト。
     - 平成元年: 福井県坂井市（旧坂井郡丸岡町）に生まれる
     - 平成20年: 福井県立藤島高校卒業
     - 平成24年: 神戸大学卒業
     - 平成26年: 同法科大学院卒業 / 司法試験合格
     - 平成27年～: 裁判所職員として約10年間勤務
     - 令和7年: 弁護士事務所に転職
     - 令和8年: 弁護士資格取得

4. **Links (外部連携)**: 
   - 関連する情報源（所属先、SNS）へ誘導するリスト。
   - 項目：株式会社フクイト、関連会社リンク、大谷尚 公式 X (旧Twitter) アカウントへのリンク枠。

5. **Contact (対話への誘導)**: 
   - 案内文：「講演・取材・プロジェクトのご依頼、AIガバナンスへの取り組みに関するご相談など」
   - 仕様：安全性を最大限考慮し、APIやフォームを使わず、ローカルのメールソフトを直接立ち上げる構成（mailtoリンク）。

## 3. 技術・デザイン方針（確約事項）
- **技術スタック**: `index.html` と `style.css` の2ファイルのみ。（スマホ対応レスポンシブデザイン済）
- **デザイン要素**:
  - **知的さとエレガンス**: ネイビブルー、ロイヤルブルーを基調とし、背景にはソフトブルー（#F0F4F8）を採用。ベージュ等の温色を排除し、専門家としての最高レベルの洗練さを表現。
  - **余白の美学**: 情報を詰め込まず、大胆に空間をとることで文字情報を際立たせます。

## 4. セキュリティ方針（遵守済）
1. **機密情報（APIキー等）の直書き禁止**: ハードコードは一切ありません。
2. **静的ファイルベースの構築**: バックエンド処理が無いため、ハッキング被害リスクは事実上ゼロです。
3. **個人情報の収集なし**: フォームを使わないため、情報漏洩のリスクを未然に防ぎます。

## 【今後の再開に向けたアクション・未完了タスク】
次回開発を再開する際は、以下の内容からスムーズに着手できます。
1. **LinksのURL設定**: Linksセクションに配置した各ダミーURLを、実際の自社URLおよびXアカウントのURLに差し替える。
2. **エッセイの推敲**: Messageセクションの仮テキストをご自身の実際の言葉へ微調整する。
3. **Contactの宛先設定**: 実際に受信したいメールアドレス（`mailto:xxx@example.com`等）を設定する。
4. **本番公開設定**: ファイルサーバーへのアップロード等による運用環境への公開。
